Moderation System

Sapphire Bot includes powerful moderation tools for administrators and moderators.

Slash Commands:

/ban – Ban a user from the server
/unban – Unban a previously banned user
/kick – Kick a user from the server
/mute – Temporarily mute a member
/unmute – Remove mute from a user
/warn – Warn a user
/warnings – View warnings of a user
/clear – Delete multiple messages
/slowmode – Set slow mode in a channel
/lock – Lock a channel
/unlock – Unlock a channel
/nickname – Change a user nickname
/timeout – Timeout a user
/untimeout – Remove timeout

Advanced Moderation:

• Mass message purge
• Auto punishment system
• Warning tracking database
• Temporary bans and mutes
• Role hierarchy protection
• Anti raid protection
